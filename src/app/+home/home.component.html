myVote The poll platform

This is a test project for modern web technologies.

Authenticated users (via Github OAuth or a fake login) can create/edit/remove their own polls. Non-autenticated users can cast a vote on any poll and view each poll's result.

All data is automatically synchronized - no browser refresh necessary. Try opening two browser instances and make a change in one browser - the other will automatically update.
